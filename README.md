# React Native Mobile Apps Portfolio

### 1. To-Do List App (`todo_list_app/`)

Task management application with add, delete, and mark-as-done functionality.

- Features: Task creation, completion tracking, deletion
- Components: Header, TaskInput, TaskList, TaskItem

### 2. Counter App (`counter_app/`)

Simple counter with increment, decrement, and reset operations.

- Features: Increment/decrement counter, reset to zero
- Components: CounterDisplay, CounterButton, ResetButton

### 3. Calculator App (`calculator_app/`)

Basic calculator with arithmetic operations and expression display.

- Features: Addition, subtraction, multiplication, division
- Components: Display, CalcButton
- Shows full expression (e.g., "5 + 3 = 8")

### 4. Quiz App (`quiz_app/`)

Multiple-choice quiz application with score tracking.

- Features: 5 sample questions, progress tracking, score calculation
- Components: ProgressBar, QuestionCard, AnswerButton, ResultsScreen

### 5. Recipe App (`recipe_app/`)

Recipe browser with search and favorites functionality.

- Features: Browse recipes, search, save favorites, view details
- Components: SearchBar, RecipeCard, RecipeDetail

### 6. Expense Tracker (`expense_tracker/`)

Daily expense management and tracking application.

- Features: Add/delete expenses, category tracking, total calculation
- Components: Summary, AddExpense, ExpenseItem
